## Setup Development Machine
1. Clone the repo `git clone git@github.com:anaida07/MEVN-boilerplate.git`

2. `cd webhealth_tracker`

3. Open client
```
cd client
npm install
npm start
```

4. Open server
```
cd server
npm install
npm start
```

5. Open `http://localhost:8080` in browser
